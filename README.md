# تطبيق الإدارة المالية (Finance App)

## 📌 وصف المشروع
تطبيق أندرويد بسيط تم تطويره كجزء من التدريب الميداني لإدارة المشاريع باستخدام Git و GitHub. يهدف التطبيق إلى مساعدة المستخدمين في تتبع نفقاتهم وإدارة ميزانيتهم اليومية بواجهة استخدام سهلة وبسيطة.

## 🛠 التقنيات المستخدمة
* **لغة البرمجة:** Java
* **بيئة التطوير:** Android Studio
* **تصميم الواجهات:** XML
* **إدارة النسخ:** Git & GitHub

## 🚀 خطوات التشغيل
لتشغيل هذا المشروع على جهازك المحلي، اتبع الخطوات التالية:
1. قم بنسخ رابط المستودع (Clone the repository):
   
   git clone(https://github.com/Eng-Heba-Alakhras/github-training-project.git)
افتح برنامج Android Studio.

اختر Open an existing project وحدد مجلد المشروع.

انتظر حتى ينتهي Gradle Build.

قم بتشغيل التطبيق على محاكي (Emulator) أو جهاز أندرويد حقيقي.
